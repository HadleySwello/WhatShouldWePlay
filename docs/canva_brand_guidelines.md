# What Should We Play? - Canva Brand Guidelines

Welcome to the brand kit for *What Should We Play?*. Use these exact HEX codes when designing your custom SVG or other visual assets in Canva to ensure perfect harmony with the application.

### The Vibe (Magician / Caregiver)
Calm, supportive, mystical, but structured. 

---

## 🎨 Core Color Palette (HEX Codes)

To add these to your Canva Brand Kit, just copy and paste the HEX codes below.

### The Foundation (Backgrounds & Cards)
These are your warm, soft base colors. Use these for large backgrounds, cards, or anywhere you want a soft touch.

*   **Powder Petal Base:** `#ECD7C7` (The absolute lightest color, used as the main bright background)
*   **Powder Petal Secondary:** `#E2CCBC` (Slightly darker for depth)
*   **Card Surface (Warm):** `#F4E7DD` 

### The Anchor (Primary Brand Colors)
This is the core identity. Use `Pine Blue` for the most important elements, and `Berry Crush` to draw the eye.

*   **Dark Pine Blue (Primary Accent):** `#2A575A` 
*   **Berry Crush (Excitement / Secondary Accent):** `#AA4465`
*   **Dark Raspberry (Deep Accent):** `#861657`

### The Void (Text & High Contrast)
Never use pure black `#000000`. Instead, use `Deep Space Blue` for your darkest elements, text, and outlines.

*   **Deep Space Blue:** `#273043`

## 📐 Design Tips for the Hero SVG

If you are designing the "What Should We Play?" header graphic:

1.  **Background:** Make sure the Canva canvas background is set to **Transparent** when you export.
2.  **Color Choice:** Since it will sit on top of the soft pastel gradient, use **Deep Space Blue (`#273043`)** or **Dark Pine Blue (`#2A575A`)** for the main text to guarantee readability.
3.  **Typography:** Try searching Canva for fonts like:
    *   *Playfair Display* (for a magical, elegant serif)
    *   *Lora* (warm and reliable)
    *   *Outfit* or *Montserrat* (clean, modern, no-nonsense)
4.  **Flourishes:** Consider adding a subtle arc, a very soft drop shadow (`#E2CCBC`), or tiny star/sparkle accents using the `Berry Crush` (`#AA4465`) color.
